## 三、申请步骤

### 步骤1：注册账号

打开网址：https://register.ccopyright.com.cn/

点击"注册"，选择"个人用户"，填写手机号、邮箱、姓名、身份证号，设置密码，完成注册。

### 步骤2：登录并进入申请页面

登录后，点击左侧菜单"我要登记"，选择"软件著作权登记申请"。

或直接访问：https://register.ccopyright.com.cn/bclLogin.html

### 步骤3：填写申请表

参照下方"申请信息填写参考"表格填写。

### 步骤4：上传材料

在"鉴别材料"页面：
- **源程序**：上传 `DualSoul_源代码文档_v2.docx`
- **文档**：上传 `DualSoul_软件说明书_v2.docx`
- **身份证明**：上传身份证扫描件

### 步骤5：提交申请

检查所有信息无误后，点击"提交"。记录申请编号（格式如：SR2026XXXXXX）。

## 七、常见问题

**Q: 要花钱吗？**
A: 官方申请免费（2017年起取消收费）。如果嫌麻烦可以找代理，费用300-500元。

**Q: 源代码会公开吗？**
A: 不会。版权保护中心对登记材料保密。

**Q: 代码已经在GitHub公开了，还能登记吗？**
A: 当然可以。软著登记和代码是否公开无关。

**Q: 审核需要多长时间？**
A: 一般30-60个工作日。加急（1-5个工作日）需额外付费。

**Q: .md格式可以直接提交吗？**
A: 不行，需要转成.docx格式。用Pandoc转换：`pandoc file.md -o file.docx`
