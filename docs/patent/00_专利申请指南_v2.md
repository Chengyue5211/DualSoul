| 项目 | 单价 | 数量 | 小计 |

| **合计** | | | **¥32,250~57,250** |

> 注：官费可申请费用减缴（个人申请人年收入<6万可减85%），减缴后约¥518/项。

### 优先级排序（预算有限时）

1. **专利1（双身份图谱）+ 专利2（四模式路由）**：保护范围最广，构成整个系统的基础
2. **专利5（TPF跨平台可移植格式）**：市场价值最高，AI分身互操作是新兴赛道
3. **专利4（人格保真翻译）**：技术独创性强，难以绕过
4. **专利3（信任证书）**：尚未完整实现，可稍后申请

### 找专利代理人

建议找**擅长软件/AI领域**的专利代理机构。准备以下材料给代理人：

1. 本文件夹中的5份专利申请材料（.md格式）
2. `docs/PATENT_DISCLOSURE.md`（技术交底书）
3. `docs/whitepaper.md`（白皮书，技术背景参考）
4. GitHub仓库地址（首次公开证据）

代理人会将这些材料改写成正式的专利申请文件（权利要求书、说明书、摘要）。
