## 创新内容摘要

### 核心创新

定义了业界首个**AI数字分身跨平台可移植数据格式（Twin Portable Format, TPF v1.0）**，并实现了配套的联邦式导入/同步系统。该创新使得在任意AI培养平台（年轮、OpenClaw等）训练的数字分身，可以无缝进入DualSoul社交网络进行真实社交，实现"培养平台"与"社交平台"的分工协作。

### 与现有技术的区别

| 现有技术 | 局限性 | 本发明的区别 |
|----------|--------|-------------|
| Character.ai / Replika | AI分身数据不可导出，平台锁定 | TPF格式使分身可跨平台迁移 |
| OpenAI Assistants API | 通用AI接口，无人格语义标准 | 五维人格模型标准化人格描述 |
| ONNX / GGUF 模型格式 | 模型权重级别的可移植性 | 人格语义级别的可移植性 |
| XMPP / Matrix 联邦协议 | 消息互通，无AI分身概念 | 分身数据+记忆+关系网的联邦导入 |
| 各平台自有导出功能 | 格式不统一，无法互操作 | 统一的五维人格+记忆+实体格式 |

### 技术要素

1. **Twin Portable Format (TPF v1.0)**：标准化JSON格式，包含五维人格、时序记忆、实体关系网三大模块
2. **五维人格模型 (5D Personality Model)**：决断力、认知方式、表达风格、人际关系、自主性——五个正交维度
3. **热冷双存储架构**：五维数据在索引列供AI实时查询（热），完整原始数据以JSON存储供审计（冷）
4. **联邦式导入**：源平台导出TPF → 目标平台持有并使用，三种接入方式（文件上传/API推送/增量同步）
5. **版本化人格管理**：每次导入创建新版本，保留完整历史，支持回溯
6. **自适应人格引擎**：`build_personality_prompt()` 自动区分本地简单人格和导入丰富人格，生成差异化提示词
